# Team Task Manager

## Project Description
A Flask-based Team Task Management web application that allows users to register, login, create projects, create tasks, and track task status.

## Features
- User Registration
- User Login Authentication
- Dashboard
- Project Management
- Task Management
- SQLite Database
- Flask Backend
- Bootstrap Frontend

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- Bootstrap
